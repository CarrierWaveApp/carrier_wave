# Add QSOs to sessions

I want to allow users to add QSOs to sessions in two ways: singular QSOs in edit, and also bulk edit. The add should open a session selector with filters for POTA/SOTA/casual, and refuse to add QSOs to new sessions that're already in sessions. Error messages must be clear, and specific--if two QSOs are selected and only one park is missing a session, we should clearly state that the specific QSO with a park can't be added. The other QSO should be added.
